关于代码框架

可以先不用管意思，直接套用即可

#include<bits/stdc++.h>
using namespace std;
int main(){
    //这里放你的代码
    return 0;
}

语法

每段代码后应加;

变量

  • 声明一个变量，名为x
    int x;
  • 声明一个变量，名为tt
    int tt;
  • 声明一个变量，名为abc
    int abc;

基本IO

cin

能够从用户输入中读取一个数，并存入指定变量
例子：

int x;
cin>>x;

或

long long y;
cin>>y;

cout

能够向用户输出指定变量的值
例子：

int x=1;
cout<<x;

或

int e;
cin>>e;
cout<<e;

或

cout<<666;

输出

一个整数表示n×10n\times 10n×10的值

输出：100
