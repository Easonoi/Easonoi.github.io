一、基本变量类型

1. 整型 int

int 是整数类型，表示没有小数点的数字，比如：

int age = 18;

上面这行代码的意思是：创建了一个名为 age 的变量，类型是整数，并把它的值设为 18。

你可以对整数变量进行数学运算：

int a = 10;
int b = 5;
int sum = a + b;  // sum 的值是 15

2. 布尔型 bool

bool 是布尔类型，只能存两个值：

  • true：表示“真”
  • false：表示“假”

布尔变量常用于判断和控制程序流程：

bool isStudent = true;

二、条件判断语句 if

if 是判断语句，它会检查一个条件是否为真（true）。如果是真的，就执行对应的代码。

例子： 

int score = 75;

if (score >= 60) {
    cout << "及格" << endl;
} else {
    cout << "不及格" << endl;
}

这段程序的意思是：

  • 如果分数大于等于 60，就输出“及格”。
  • 否则输出“不及格”。

if 基本结构： 

if (条件) {
    // 条件为真时执行的代码
} else {
    // 条件为假时执行的代码（可选）
}

条件里可以使用比较运算符，比如：

运算符 含义 示例
== 等于 x == 10
!= 不等于 x != 5
> 大于 x > 3
< 小于 x < 8
>= 大于等于 x >= 90
<= 小于等于 x <= 60

三、自操作运算符

在程序中，经常需要对变量进行加减，比如把某个数加 1 或减 2。C++ 提供了简化写法，称为“自操作运算符”。

1. 自增运算符 ++

++ 是“加 1”的快捷方式：

int x = 5;
x++;     // x 变成 6，等价于 x = x + 1;
++x;     // 再变成 7
  • x++ 是“后缀自增”：先使用变量，再加 1
  • ++x 是“前缀自增”：先加 1，再使用变量

对初学者来说，两种写法在单独使用时效果一样。

2. 自减运算符 --

类似地，-- 表示将变量减 1：

int y = 3;
y--;     // y 变成 2
--y;     // y 变成 1

3. 复合赋值运算符

有时候我们要对变量进行加减乘除并赋值，C++ 提供了更简洁的写法：

运算符 含义 示例 等价于
+= 加后赋值 a += 3; a = a + 3;
-= 减后赋值 a -= 2; a = a - 2;
*= 乘后赋值 a *= 4; a = a * 4;
/= 除后赋值 a /= 2; a = a / 2;
%= 取余后赋值 a %= 3; a = a % 3;

示例： 

int a = 10;
a += 5;   // 变成 15
a *= 2;   // 变成 30
a -= 10;  // 变成 20

四、循环结构

程序中常常需要重复做某件事，比如输出一段文字多次，这时候就需要用“循环”。

1. while 循环

只要条件满足，就一直执行代码：

int i = 0;
while (i < 5) {
    cout << i << endl;
    i++;
}

程序解释：

  • i 从 0 开始
  • 每次循环打印 i
  • i++ 表示每次加 1
  • 当 i 到 5 时，条件不成立，循环结束

2. for 循环

和 while 类似，但结构更紧凑，常用于已知循环次数的情况：

for (int i = 0; i < 5; i++) {
    cout << i << endl;
}

这和前面的 while 做的是一样的事情。

结构说明：

for (初始化; 条件; 每次循环后的操作) {
    // 循环体
}

continue 语句

continue 用于跳过本次循环中剩下的代码，直接进入每次循环后的操作。

举例： 

for (int i = 1; i <= 5; i++) {
    if (i == 3) {
        continue;  // 跳过 i = 3，不执行下面的 cout
    }
    cout << i << " ";
}

输出结果是：1 2 4 5

解释：

  • 当 i == 3 时，continue 使程序跳过 cout，跳转到i++语句，并开始下一轮循环

break 语句

用来提前结束循环：

for (int i = 0; i < 10; i++) {
    if (i == 5) {
        break;  // 当 i 是 5 时跳出循环
    }
    cout << i << endl;
}

运行结果是输出 0 到 4，当 i 等于 5 就不再继续循环。

小结

  • int 存整数，bool 表示逻辑值
  • if 判断条件，决定程序执行哪段代码
  • ++, --, += 等运算符可让代码更简洁
  • 使用 while 和 for 循环来重复执行代码
  • break 可用于提前跳出循环
  • continue 跳到

注意事项

1.

以下代码不合法：

for(int i=1;i<=5;i++){cout<<i;}
cout<<i;

因为i只在循环内部被定义，跳出循环时会被删除（或者说销毁）

2.

以下代码合法：

int i=1145;
for(int i=1;i<=5;i++){cout<<i;}
cout<<i;

这里涉及到就近原则：程序会使用最内部定义的变量，无视外部的同名变量
不过不推荐这种写法，容易混淆变量

3.代码示例 

int c=0;
for(int i=1;i<=10;i++){
    int val;
    cin>>val;
    if(val==1)c++;
}
cout<<c;

这段代码实现了输入10个数字，统计其中1的个数，其中val变量只在循环内部有效，每次循环时，被重新定义
